Instructions
CSV Format

First row must be headers. Examples:

Machines: 
name,model,serial_number,machine_number
Note: site_id will be automatically set to selected site
Parts: 
name,description,machine_name,maintenance_frequency,maintenance_unit
Note: machine_name should match existing machine names in the selected site
Maintenance Records: 
machine_name,part_name,maintenance_type,description,date,performed_by,status,notes
Note: machine_name and part_name should match existing records in the selected site
JSON Format

Array of objects. Examples:

Machines: 
[
  {"name": "Machine 1", "model": "X100", "serial_number": "SN123"},
  {"name": "Machine 2", "model": "X200", "serial_number": "SN124"}
]
Parts: 
[
  {"name": "Motor", "description": "Main motor", "machine_name": "Machine 1", "maintenance_frequency": 30, "maintenance_unit": "day"},
  {"name": "Belt", "description": "Drive belt", "machine_name": "Machine 2", "maintenance_frequency": 60, "maintenance_unit": "day"}
]
